# Code of Conduct

## Scope

This code of conduct applies to anyone who participates in the oxiflow project or who speaks
publicly about oxiflow — in the repository, in issues, in pull requests, in discussions, in
third-party `oxiflow-*` framework repositories, and in any public space where oxiflow is a
subject.

## What we expect

- Be respectful. Disagreement is fine; contempt is not.
- Criticise ideas, not people.
- Be precise. Vague complaints are not useful to anyone.
- If you don't understand something, ask. If you know something, share it.

## What is not acceptable

- Personal attacks, insults, or demeaning remarks directed at contributors.
- Harassment of any kind, in public or in private.
- Dismissing contributions without explanation.
- Deliberate disruption of discussions.

## Reporting

If you witness or experience behaviour that violates this code of conduct, report it privately
to the maintainer. Reports are handled confidentially.

Contact: private email — address available in the repository's `Cargo.toml` or the
maintainer's GitHub profile.

## Enforcement

The maintainer will investigate reports and may remove comments, close issues, or exclude
contributors who repeatedly or seriously violate this code of conduct.

## Basis

This code of conduct is adapted from the
[Rust Code of Conduct](https://www.rust-lang.org/policies/code-of-conduct).
