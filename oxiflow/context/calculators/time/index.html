§Module context::calculators::time

Built-in calculators for temporal context variables.

TimeCalculator and TimeStepCalculator are injected by the solver before the user-defined calculator chain runs. They are registered here as public types so that downstream models can inspect or test the chain without coupling to solver internals.

Structs§

TimeCalculator
Provides the current simulation time as ContextVariable::Time.
TimeStepCalculator
Provides the current time step as ContextVariable::TimeStep.
