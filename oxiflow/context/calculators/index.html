§Module context::calculators

Built-in ContextCalculator implementations for common PDE quantities.

These calculators cover the most frequent context variables in transport–reaction–diffusion problems. Each implements ContextCalculator and can be registered directly in SolverConfiguration.

§Available calculators

TypeProvidesNotes
TimeCalculatorContextVariable::TimeCurrent simulation time
TimeStepCalculatorContextVariable::TimeStepCurrent dt

§Mesh ownership

Spatial calculators (FDGradientCalculator, FDLaplacianCalculator, TrapezoidalIntegral) hold an Arc<dyn Mesh> internally. This is an implementation detail consistent with INV-1 (DD-007): the mesh never appears in the public ContextCalculator API. At v0.5.0, Arc<dyn Mesh> will be replaced by a concrete DiscreteOperator (INV-2, DD-012), with zero change to the ContextCalculator trait signature.
