§Module solver::config

Solving configuration — SolverConfiguration (HOW pole, DD-021, issue #32).

§Design

dt: f64 is never exposed directly in the public API. Instead, StepControl encapsulates both fixed-step and adaptive-step strategies. TimeConfiguration groups all temporal parameters. This prevents the breaking change that would occur at J4 when adaptive integrators (DoPri45, BDF2) are introduced (DD-021).
