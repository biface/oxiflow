§Module solver::scenario

Problem declaration — Scenario (WHAT pole, DD-020, issue #32).

§Design

Scenario is unified: it handles both single-domain (J1) and multi-domain (J3) problems. The single-domain case is the degenerate case with one element in domains and empty couplings. Ergonomic helpers cover the J1 use case without verbosity. At J3, additional domains and coupling operators are added without any API change (DD-020).
