§Module mesh

Spatial mesh abstraction — INV-1 invariant (DD-007, DD-019, issue #31).

§Core principle (INV-1)

No public engine API exposes dx, nx or raw indices as first-class spatial parameters. All spatial references go through the Mesh trait. This guarantees forward compatibility with FEM unstructured meshes at J7 — zero breaking change on existing code when unstructured/ is added.

§Module hierarchy (DD-019)
